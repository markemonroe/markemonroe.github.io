ParallelProjection
==============================================================================
Utility class to construct a parallel projection matrix for use in a camera.  The size, near and far attributes of a camera have the same
meaning as the attributes in this class.  To change only certain attributes of a projection, the remaining other attributes can be set from
an existing camera.
