EdgeSelectionFilter
==============================================================================
Filter for selection of an edge of a :py:class:`CADModel<geomagic.api.v3.swig.CADModel>`.  Returns the edge id as an int.
Used in :py:class:`EdgeSelector<geomagic.app.v3.gui.selector.EdgeSelector>`.
See :py:class:`Selector<geomagic.app.v3.gui.selector.Selector>` or :py:class:`PointSelectionFilter<geomagic.app.v3.gui.selector.PointSelectionFilter>` for an example on how to use selection filters.
