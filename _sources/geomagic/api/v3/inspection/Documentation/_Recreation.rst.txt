Recreation
==============================================================================
Parent class that defines the autocreation information of Feature data objects (Feature3DRecreationInfo, Feature2DRecreationInfo, or TargetRecreationInfo).  Note that these parameter influence *only* the autocreation of the feature, that is they do not change thegeometry of the feature itself.
