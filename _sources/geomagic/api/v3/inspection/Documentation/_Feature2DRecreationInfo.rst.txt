Feature2DRecreationInfo
==============================================================================
Defines the autocreation information for *planar* Feature data objects(HoleFeature, OvalSlot, RoundedSlot, and RectangularSlot). Note that these parameter influence *only* the autocreation of the feature, that is they do not change the geometry of the feature itself.
