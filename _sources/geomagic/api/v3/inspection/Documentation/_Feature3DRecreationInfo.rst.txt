Feature3DRecreationInfo
==============================================================================
Defines the autocreation information for *spatial* Feature data objects (Plane, Cylinder, Cone, Sphere, and ParallelPlanes).  Note that these parameter influence *only* the autocreation of the feature, that is they do not change the geometry of the feature itself.
