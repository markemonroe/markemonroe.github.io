CreateTextureFromColorMap
==============================================================================
Utility to create a :py:class:`Texture<Texture>` from a :py:class:`ColorMap<ColorMap>`.

This tool generates texture maps that can be used in :py:class:`Mesh<Mesh>` or :py:class:`VertexObject<VertexObject>` shapes.
