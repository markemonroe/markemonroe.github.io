CreateTextureCoordinatesFromColorMap
==============================================================================
Utility class to create an array of texture coordinates from values and a :py:class:`ColorMap<ColorMap>`.

This tool generates texture coordinates that can be used in :py:class:`Mesh<Mesh>` or :py:class:`VertexObject<VertexObject>` shapes.
