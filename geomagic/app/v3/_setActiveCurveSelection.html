setActiveCurveSelection¶

setActiveCurveSelection(crv, crvSelection)¶

Examples

Load a test curves model, make a selection of all curves on the object, and make that selection active. This will make the selection red.

import geomagic.app.v3
from geomagic.app.v3.imports import *
      
# Open test file
path = Directories.getDirectory(Directories.DIRECTORY_EXAMPLE_DATA)
geoapp.openFile(path + "\\curves.wrp")

# Get the active model.
activeModel = geoapp.getActiveModel()

# Get the curve from the active model.
curves = geoapp.getCurveSet(activeModel)

# Select all faces on the curve set
curveSelection = CurveSelection(curves)
curveSelection.selectAll()

# Set the new selection as the active selection
geoapp.setActiveCurveSelection(curves, curveSelection)
