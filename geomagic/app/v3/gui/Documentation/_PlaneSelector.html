PlaneSelector¶

Convenience class to pick a plane. Different types of planes can be selected:

  • On CAD models: Planar faces

  • On features: For plane features, this is the feature itself. For higher level features, derived planes can be selected too, such as the end planes of a cylinder, etc.

  • On coordinate systems: a principal plane of a coordinate system. The plane is oriented in such a way that its normal faces the positive direction.

  • On world axes: a principal plane of the world. The plane is oriented in such a way that its normal faces the positive direction.

PlaneSelector is implemented using Selector, where its filter is PlaneSelectionFilter.

Examples

import geomagic.app.v3
from geomagic.app.v3.imports import *
class SelectionDemo(guiDlg.DialogHandler):

   def __init__(self, name, modal):
      super(SelectionDemo, self).__init__(name, modal)
   
   def formDialog(self):
      self.dialog().setButtonVisible (1, False);
      self.dialog().setButtonVisible (2, False);
      return True

   def onStart(self):
      self.selector = guiSelector.PlaneSelector()
      self.selector.planeSelected += self.onPlaneSelected
      return True
     
   def onStop(self):
      self.selector.stop()
     
   def onPlaneSelected(self, sender, plane):
      print("plane origin: {0} normal: {1}".format(plane.origin, plane.normal))

   def onOK(self):
      return True
     
demo = SelectionDemo(u'Selection Demo', False)
demo.start()
