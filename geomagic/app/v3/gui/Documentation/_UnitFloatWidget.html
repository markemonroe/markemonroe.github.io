UnitFloatWidget¶

Presents an edit box for inputting double values. The double value is available by calling the method getValue() and can be set using the setValue() method. Value change notification can be received by subscribing to the valueChanged(sender, value, reason) event.

class UnitFloatWidget(id, label)¶

Bases: geomagic.app.v3.guiV3.GeoUnitFloatWidget

Methods

__init__(self, id, label) → UnitFloatWidget¶
onValueChanged(value, reason)¶

Called by the GUI system to notify listeners that the value has changed. The new value is in value and the

name of the trigger for this even is in reason. Valid Reasons are: EnterPressed (user pressed the Enter key),

LostFocus (user clicked on another widget), WheelChange (user moved the mouse wheel),

ArrowChange (user pressed an arrow key), RMBChange (user pressed mouse button 3),

ValidatorChange (the dialog executed an input validation cycle and changed the input to conform). Add a

listener to valueChanged(sender, value, reason) to receive notification that the text content has changed.

Examples

Create and show a simple dialog with an unit float widget.

import geomagic.app.v3
from geomagic.app.v3.imports import *

class UnitFloatWidgetDialogHandler(guiDlg.DialogHandler):
   
   # Constructor. You must call the base class constructor.
   def __init__(self, name, modal):
      super(UnitFloatWidgetDialogHandler, self).__init__(name, modal)
      self.dlg = self.dialog()
      
   # All Widgets must be created in formDialog.
   def formDialog(self):
      
      # Get the top Frame to add widgets to.
      topFrame = self.dialog().topFrame()
      
      # Create a UnitFloatWidget and add it to the top frame.
      fw1 = guiWidgets.UnitFloatWidget('fw1Ref', u'This is a Float Widget')
      topFrame.addWidget(fw1)
      
      # Subscribe to the value changed event by adding a method to call.
      fw1.valueChanged += self.onValueChanged
      
      # Set the range of the widget.
      fw1.setLimits(0.0, 1000000000.0, 10.0)
      
      return True
   
   # This method is called when the value is changed. Please note that the value returned here is in meters.
   def onValueChanged(self, sender, value, reason):
      print("The new value is: " + str(value))
      
# Now start the dialog
dialog = UnitFloatWidgetDialogHandler('Unit Float Widget Dialog', False)
dialog.start()
