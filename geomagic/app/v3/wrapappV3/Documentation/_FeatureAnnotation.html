FeatureAnnotation¶

A comparison between a nominal and measured feature. Not all properties are applicable for all feature types. If a property is not applicable to a given feature, it will throw an error. All properties are read only.

viewName (string) The name of the view that contains the annotation

form (double) A measurement of the form of the feature

formTolerance (double) The tolerance that determines pass/fail of form

normalNominal (GeoVector) The normal of the reference feature

normalMeasured (GeoVector) The normal of the test feature

normalDeviation (double) Deviation between nominal/measured normal

normalDeviationUppwerTolerance (double) Upper limit of the normal deviation

normalDeviationLowerTolerance (double) Lower limit of the normal deviation

centerNominal (GeoVector) The center of the reference feature

centerMeasured (GeoVector) The center of the test feature

centerDeviation (double) Deviation between nominal/measured center

centerDeviationUpperTolerance (double) Upper limit of the center deviation

centerDeviationLowerTolerance (double) Lower limit of the center deviation

diameterNominal (double) The diameter of the reference feature

diameterMeasured (double) The diameter of the test feature

diameterDeviation (double) Deviation between nominal/measured diameter

diameterUpperTolerance (double) Upper limit of the diamter deviation

diameterLowerTolerance (double) Lower limit of the diameter deviation

axisNominal (GeoVector) The axis of the reference feature

axisMeasured (GeoVector) The axis of the test feature

axisDeviation (double) Deviation between nominal/measured axis

axisDeviationUpperTolerance (double) Upper limit of the axis deviation

Examples

Get all the measurements from the active model

Get all the measurements from the active model

import geomagic.app.v3
from geomagic.app.v3.imports import *

# get the active model and find all measurements

import geomagic.app.v3
from geomagic.app.v3.imports import *
from geomagic.app.v3.wrapappV3 import *

model = geoapp.getActiveModel()
gm = GetMeasurementsFromModel()
gm.model = model
gm.run()

for m in gm.measurements:
    print ("name: " + m.name)
    print ("value: " + str(m.value))
    print ("tolerance: " + str(m.tolerance))
    if m.result == Measurement.PASS:
        print ("status: PASS")
    elif m.result == Measurement.FAIL:
        print ("status: FAIL")
    else:
        print ("status: INVALID")

    if isinstance(m, Dimension):
        print ("Dimension:")
        print ("    viewName: " + m.viewName)
        print ("    dimType: " + m.dimensionType)
        print ("    measured: " + str(m.measured))
        print ("    nominal: " + str(m.nominal))
        print ("    upperTol: " + str(m.upperTolerance))
        print ("    lowerTol: " + str(m.lowerTolerance))
    elif isinstance(m, GDTCallout):
        print ("Callout:")
        print ("    viewName: " + m.viewName)
        print ("    dimType: " + m.dimensionType)
        print ("    bonusTolerance: " + str(m.bonusTolerance))
        print ("    minimumValue: " + str(m.minimumValue))
        print ("    maximumValue: " + str(m.maximumValue))
    elif isinstance(m, DeviationAnnotation):
        print ("Annotation:")
        print ("    viewName: " + m.viewName)
        print ("    upperTol: " + str(m.upperTolerance))
        print ("    lowerTol: " + str(m.lowerTolerance))
        print ("    deviationRadius: " + str(m.deviationRadius))
        print ("    normal: " + str(m.normal))
        print ("    measured: " + str(m.measured))
        print ("    nominal: " + str(m.nominal))
    elif isinstance(m, FeatureAnnotation):
        print ("Feature:")
        print ("    viewName: " + m.viewName)
        print ("    featureType: " + m.featureType)
        print ("    form: " + str(m.form))
    elif isinstance(m, WallthicknessAnnotation):
        print ("Wallthickness:")
        print ("    viewName: " + m.viewName)
        print ("    upperTol: " + str(m.upperTolerance))
        print ("    lowerTol: " + str(m.lowerTolerance))
        print ("    measured: " + str(m.measured))
        print ("    nominal: " + str(m.nominal))
