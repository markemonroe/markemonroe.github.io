Model¶

This is an object that appears in the ModelManager in the application. Generally this will have a Points, GriddedPoints, or Mesh object as a child and may have features as well. When Meshes, Points, and GriddedPoints objects get added to the application a Model object is created to hold them.

The following prim strings are supported for get and set prim visible.

“PrimEdges” “PrimBoundaryEdges” “PrimNonManifoldEdges” “PrimTriangles” “PrimColorBackfacingTriangles” “PrimMarkedEdges” “PrimMarkedVertices” “PrimViewCurvatureMap” “PrimWireFrame” “PrimTriWireFrame” “PrimHalfDiscs” “PrimVertexMap” “PrimWireFrameCorners” “PrimPatchBoundaries” “PrimCharacterLines” “PrimMarkedEdgeLines” “PrimBoundaryLines” “PrimExtendedLines” “PrimPoorPatchAngles” “PrimLowDegreeCorners” “PrimHighDegreeCorners” “PrimShadedPatches” “PrimGrids” “PrimShadedGrids” “PrimFlowLines” “PrimSpecifiedSharpLines” “PrimFlatSurfaces” “PrimSpaceGrids” “PrimSpaceGridTestMesh” “PrimControlPointTestMesh” “PrimControlPoints” “PrimNurbsIsoParms” “PrimSplinePatches” “PrimSplineBoundaries” “PrimSplineTestCurves” “PrimSplineTestCurveCPs” “PrimSplineTestSurfaces” “PrimSplineTestSurfaceCPs” “PrimSplineNormals” “PrimSplineCurveFitNormals” “PrimSplinePrinCurv1” “PrimSplinePrinCurv2” “PrimSplineCurvature” “PrimSplineNonG1Verts” “PrimSplineNonG1VertVectors”

Load a mesh object and turn on edges.

import geomagic.app.v3
from geomagic.app.v3.imports import *
# Open test file
path = Directories.getDirectory(Directories.DIRECTORY_EXAMPLE_DATA)
geoapp.openFile(path + "\\axial.obj")

# Get the currently selected model in the model manager.
activeModel = geoapp.getActiveModel()

# Turn on edges.
activeModel.setPrimVisible("PrimEdges", True)

Transform Model around the x-axis 90 degrees.

import geomagic.app.v3
from geomagic.app.v3.imports import *
import math

# Open test file
path = Directories.getDirectory(Directories.DIRECTORY_EXAMPLE_DATA)
geoapp.openFile(path + "\\axial.obj")

# Get the currently selected model in the model manager.
activeModel = geoapp.getActiveModel()

# Transform model.
t = Transform3D()
t.setRotationTransform(Vector3D(1.0, 0.0, 0.0), math.pi * 0.5)

activeModel.transform(t);

Create two point objects and add them to a new group.

import geomagic.app.v3
from geomagic.app.v3.imports import *

pts1 = Points()
pts1.addPoint(Vector3D(1,0,0))
pts1.addPoint(Vector3D(0,1,0))
pts1.addPoint(Vector3D(0,0,1))

if sys.version_info.major == 3:
   addModel = geomagic.app.v3.appdataV3.AddModel()
else:
   addModel = geomagic.app.v3.appdataV3.AddModel()
addModel.name = u"Points 1"
addModel.model = pts1
addModel.run()
model1 = addModel.object

pts2 = Points()
pts2.addPoint(Vector3D(6,0,0))
pts2.addPoint(Vector3D(5,1,0))
pts2.addPoint(Vector3D(5,0,1))

addModel.name = u"Points 2"
addModel.model = pts2
addModel.run()
model2 = addModel.object

group = geoapp.newGroup(u"Point Group")
model1.addToGroup(group)
model2.addToGroup(group)
for model in group.groupModels:
   print model.name
