RegionOfInterest¶

A utility class that constructs an orientation and suitable zoom factor based on selections on Points, GriddedPoints, Mesh or Feature objects. Alternatively, entire objects can be used.

class RegionOfInterest¶

Bases: geomagic.api.v3.swig.Modifier

Attributes

Usage hints

Description

camera

No

DataObject

None

The camera to modify. This modifier will keep the projection, but replace the orientation and zoom factor.

directionHint

No

Vector3D

[0.00 0.00 0.00]

Sets the view direction if none of the models implicitly provided a view direction using their normals. Points and Features may not provide such normal information.

directionOverride

No

bool

False

If set, always use the directionHint, regardless of the local surface normals found on the models.

ignoreSelection

No

bool

False

If set, the full model is used, and the selection is ignored.

name

No

string

None

Name of the modifier.

up

No

Vector3D

[0.00 0.00 0.00]

Vertical (“up”) direction in the view of the camera. If this is not set, the existing up direction will be kept.
