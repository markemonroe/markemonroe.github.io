Correspondence¶

Represents a correspondence between entities in a test object and a reference object. Valid combinations are: Points-Mesh, Points-CAD, Mesh-CAD. A Correspondence object should not be constructed directly, use CreateCorrespondence modifier to create a new Correspondence object.

Examples

Create Correspondence between Mesh and Points and get points which are bound to the first mesh triangle.

from geomagic.api.v3 import *

# Create mesh and points
mesh = openCone()
ptCover = CreatePointCover(mesh)
ptCover.run()
points = ptCover.points

# Create Correspondence
creator = CreateCorrespondence()
creator.testObj = points
creator.refObj = mesh
creator.run()
correspondence = creator.correspondence

# Get ID of a first mesh triangle
tsel = TriangleSelection(mesh)
tsel.selectAll()
refId = tsel.first()

# Get points which correspond to the first reference triangle
ptsel = correspondence.getSelection(refId, Correspondence.Type_Triangle)
