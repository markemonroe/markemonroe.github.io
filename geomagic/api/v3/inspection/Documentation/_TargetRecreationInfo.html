TargetRecreationInfo¶

Defines the autocreation information for target Feature data objects (TargetPoint and TargetLine). Note that these parameter influence only the autocreation of the feature, that is they do not change the geometry of the feature itself.

Determines the length of the TargetLine, that is the section where the line can touch the model. If the point of contact lies outside the segment determined by this quantity then it is discarded. Note that this is sensitive if finiteLength is True. It is meaningful for TargetLine features only.

Determines the distance by which the starting (nominal) point of the feature is pull backed to emanate a ray towards the model (in the direction of the contactDirection. It is meaningful for TargetPoint features only.

Constants

Name

Description

TargetLine_1PtCylinder

This parameter may be passed to the contactArea property of the recreation info of a of a TargetLine. It means that the target line feature is to be autocreated as if a cylinder shaped area touched the model in at least one point. The width of the band is size of the recreation info or if it is not given then it will be the size of the target line.

TargetLine_2PtCylinder

This parameter may be passed to the contactArea property of the recreation info of a of a TargetLine. It means that the target line feature is to be autocreated as if a cylinder shaped area touched the model in at least two (!) point. The cylinder is capable of moving in a perpendicular direction woth respect to the plane of the referenence target line. The width of the band is size of the recreation info or if it is not given then it will be the size of the target line. Not yet implemented (in v3 the autocreation works as a one-point cylinder (TargetLine_1PtCylinder)).

TargetLine_Band

This parameter may be passed to the contactArea property of the recreation info of a of a TargetLine. It means that the target line feature is to be autocreated as if a band shaped planar area touched the model. The width of the band is size of the recreation info or if it is not given then it will be the size of the target line.

TargetLine_Slice

This parameter may be passed to the contactArea property of the recreation info of a of a TargetLine. If the model is a Mesh or a CADModel then the model will be computed as an intersected as intersection with a plane. Otherwise (if the model is a Points) then it works as a TargetLine_Band.

TargetPoint_Disk

This parameter may be passed to the contactArea property of the recreation info of a TargetPoint. It means that the target point feature is to be autocreated as if a circular disk shaped planar area touched the model. The diameter of the disk is size or if it is not given then twice the radius of the target point.

TargetPoint_Ray

This parameter may be passed to the contactArea property of the recreation info of a TargetPoint. If the model is a Mesh then the model will be intersected with a zero diameter ray. Otherwise (if the model is a Points) then it works as a TargetPoint_Disk. That is the target point feature is to be autocreated as if a circular disk shaped planar area touched the model. The diameter of the disk is size or if it is not given then twice the radius of the target point.

Assigns some values to properties and checks their validity.

# Create recreationinfo

tg.length = 0.05
tg.finiteLength = True
#print ("Length: " + str(tg.length) + " finite:" + str(tg.finiteLength))
car = tg.contactArea
#print ("Contact area:" + str(car))
