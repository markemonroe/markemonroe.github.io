stitchingMode

No

int

0 - 2

1

Input (optional)

Specifies the ACIS stitching style to be performed. It has no effect if the kernel is not ACIS. It can be Tolerant (default) or RestrictedTolerant. In the Tolerant option the geometric proximity decides whether two topological elements are stitched or not. If the body has many edges this can be slow, but perhaps it is more reliable than the RestrictedTolerant method. The algorithm tries to bridge ALL gaps, even large ones. The RestrictedTolerant option is an edge based stitching mode. It means that ONLY edges are stitched and ONLY THOSE that are explicitly identified in the GMCAD model. This means better performance, and that open surface shells can be produced. On the other hand there might be more edges left open after stitching and some models may fall apart to disconnected components.
