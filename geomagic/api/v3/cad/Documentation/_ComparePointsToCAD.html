ComparePointsToCAD¶

Compares a reference CAD model and a test points object. To compare two CAD objects, convert the test object to Points using CreatePointsFromMesh or PointCover. The result is represented as CADCompareInfo object.

Examples

Create a point cover of a CAD model and compare it against the mesh.

from geomagic.api.v3 import *
from geomagic.api.v3.cad import *

# Create CAD model from sphere feature
sphere = Sphere()
sphere.initialize(Vector3D(0.0, 0.0, 0.0), 1.0)

converter = CreateCADFromSurface()
converter.surface = sphere
converter.run()
model = converter.cadModel
mesh = model.mesh

# Create a point cover over model mesh
ptCover = CreatePointCover(mesh)
ptCover.generateNormals = True
ptCover.offset = 0.001
ptCover.targetNumPoints = 10000
ptCover.run()
p = ptCover.points

# Compare points in p against the CAD model
compare = ComparePointsToCAD(model)
compare.points = p
compare.maxDeviation = 0.01
compare.spacing = 0.25
compare.comparisonType = compare.ComparisonType_Regular3D
compare.algorithmType = compare.AlgorithmType_Dense
compare.criticalAngle = 45.0
compare.run()

# The result is represented by a Compare3DInfo type object
comp3dinfo = compare.comparisonResult
