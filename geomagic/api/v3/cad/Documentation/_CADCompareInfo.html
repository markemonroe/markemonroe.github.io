CADCompareInfo¶

Represent result of CAD to points comparison; this class is a compact array. It is created by ComparePointsToCAD modifier. Various result properties (distance, closest point, face index etc.) can be queried by index of corresponding point in original Point object.

Examples

Create an CADCompareInfo object by running ComparePointsToCAD.

from geomagic.api.v3 import *
from geomagic.api.v3.cad import *

# Create CAD model from sphere feature
sphere = Sphere()
sphere.initialize(Vector3D(0.0, 0.0, 0.0), 1.0)

converter = CreateCADFromSurface()
converter.surface = sphere
converter.run()
model = converter.cadModel
mesh = model.mesh

# Create a point cover over model mesh
ptCover = CreatePointCover(mesh)
ptCover.generateNormals = True
ptCover.offset = 0.001
ptCover.targetNumPoints = 1000
ptCover.run()
p = ptCover.points

# Compare points in p against the CAD model
compare = ComparePointsToCAD(model)
compare.points = p
compare.maxDeviation = 0.01
compare.spacing = 0.25
compare.comparisonType = compare.ComparisonType_Regular3D
compare.algorithmType = compare.AlgorithmType_Dense
compare.criticalAngle = 45.0
compare.run()

# The result is represented by a Compare3DInfo type object
comp3dinfo = compare.comparisonResult
maxdev = comp3dinfo.maxDistance
mindev = comp3dinfo.minDistance
avdev = comp3dinfo.averageDistance
stddev = comp3dinfo.stdDevDistance
comparedPts = comp3dinfo.comparedPointSelection
for i in comparedPts:
   pt2 = comp3dinfo.getPoint(i);
   pt2proj = comp3dinfo.getClosestPoint(i);
   dist_2 = pt2.dist(pt2proj)
