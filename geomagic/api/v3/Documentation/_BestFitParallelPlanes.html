BestFitParallelPlanes¶

Creates a parallel planes feature based on a selection on a mesh, griddedpoints, or points model.

class BestFitParallelPlanes¶

Bases: geomagic.api.v3.swig.FeatureFitter

Attributes

Name

Read only

Type

Range

Default Value

Usage hints

Description

selectionArray

No

DataObjectArray

Input (required)

This is a DataObjectArray consisting of two Selection-s. The Selection-s should be either TriangleSelection-s, or PointSelection-s, or QuadSelection-s and they must refer to the SAME object (they must represent the subset of the same set).

constraint

No

FittingConstraint

None

Input (optional)

An optional fitting constraint.

contactFitting

No

bool

False

Input (optional)

False means that the surface will run in between, as an average of the points defining the surface/feature, True means it runs at the outside boundary of the set of the points (outDir determines which side the material is).

extentOutlier

No

double

-INF - +INF

0.001000

Input (optional)

0 <= extentOutlier < 1. This is used for establishing the extent of the feature. (Size of rectangle of area of plane.) This will be the ratio of points at the upper and lower end tails that will be cut off as outliers to establish the size of the feature.

robustFitIterations

No

int

-INF - +INF

2

Input (optional)

Set it zero if no automatic outlier removal is needed. If it is non-zero then it specifies the number robust fit iterations to be performed during the fitting. (See robustFitOutliers.)

robustFitOutliers

No

double

-INF - +INF

20.000000

Input (optional)

Percentage (0 <= ro < 100). For outlier removal in best fit algorithms the user can request robust fit iteration. This means that after a full best-fit least-squares fit, all the points are reevaluated as distance from the initial feature and the farthest ones of robustFitOutliers percentage are (temporarily) discarded. Then a refit is performed for the remaining points. This is done robustFitIterations times.

resultFeature

No

ParallelPlanes

None

Output

The output ParallelPlanes.

Constants

Name

Description

HighPoint

If contactFittingMode is HighPoint then an optimized algorithm is activated that creates the limiting feature.

Hollow

If solidity is Hollow then the feature will be considered a void (a slot). The contact feature is a largest limiting feature that is enclosed by the selection.

NonSize

If solidity is NonSize then the orientation is determined by the normals or, if they do not exist, the feature will be considered solid.

Solid

If solidity is Solid then the feature will be considered a solid (a slab). The contact feature is a smallest limiting feature that contains the selection.

TransBestFit

If contactFittingMode is TransBestFit then a best-fit (least-squares) parallel planes feature will be created and it will be shifted (translated) in such a way that it passes through the outermost point.

Examples

Fits a pair of parallel planes to two components of sample set of points.

from geomagic.api.v3 import *
# create a plane to sample
plane1 = Plane()
plane1.normal = Vector3D(0.0, 0.0, 1.0)          
# create sample points from plane
featureSampler = CreatePointSelectionFromSurface()
featureSampler.normalNoise = 0.0001
featureSampler.numberOfSamples = 50
featureSampler.surface = plane1
featureSampler.normal = True
featureSampler.run()
# Whole Points object
fullPts = Points()
#
ptSelection1 = PointSelection(fullPts)
for i in featureSampler.points :
   j = fullPts.addPoint(featureSampler.points.getPosition(i))
   ptSelection1.select(j)
# create another plane to sample
plane2 = Plane()
plane2.normal = Vector3D(0.0, 0.0, 1.0) 
plane2.origin = Vector3D(0.0, 0.0, 0.1)
# create sample points from plane
featureSampler2 = CreatePointSelectionFromSurface()
featureSampler2.normalNoise = 0.0001
featureSampler2.numberOfSamples = 100
featureSampler2.surface = plane2
featureSampler2.normal = True
featureSampler2.run()
#
ptSelection2 = PointSelection(fullPts)
for i in featureSampler2.points :
   j = fullPts.addPoint(featureSampler2.points.getPosition(i))
   ptSelection2.select(j)
#
selList = DataObjectArray()
selList.append(ptSelection1)
selList.append(ptSelection2)
# Best fit based on these two selections
bestFitPP = BestFitParallelPlanes()
bestFitPP.selectionArray = selList
bestFitPP.run()
# Parallel Planes feature created by the fitter 
pp = bestFitPP.resultFeature
