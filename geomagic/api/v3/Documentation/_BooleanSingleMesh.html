BooleanSingleMesh¶

Perform mesh boolean on two components of the mesh. Empty component is assumed to be complimentary to set component.

Examples

Perform boolean intersections of two meshes using MeshMerge and BooleanSingleMesh.

from geomagic.api.v3 import *

# create two copies of a sample mesh and position them differently.
m = createRefinedTetrahedron()
m1 = createRefinedTetrahedron()
rotvec = Vector3D(0, 0, 1)
rotmat = Transform3D()
rotmat.setRotationTransform(rotvec, 0.25)
m1.transform(rotmat)

# merge mesh m1 to m
merger = MergeMeshes(m)
merger.meshModelAdd = m1
merger.run()

# creates two selections corresponding to the original m and m1
selector = DecomposeSelection(m)
selector.run()
selections = selector.components

# run boolean intersection
intersector = BooleanSingleMesh(m)
intersector.firstTriSelection = selections[0]
intersector.secondTriSelection = selections[1]
intersector.operation = intersector.Union
intersector.run()
