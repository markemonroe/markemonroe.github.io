TextureÂ¶

A color texture map that can be applied to shapes.

For internal reasons, the width and height of a texture is always a power of 2. If images of different dimensions are assigned to a texture, they are expanded to the next higher power of two to satisfy this requirement. The imageHeight and imageWidth attributes can be used to query the original dimensions of the image.

Textures support transparency using the alpha channel of colors. Note that for transparency to be effective, the material that is assigned to a shape must have alpha values of less than 1.

Examples

Create and initialize sample texture, write color to specific texels and read it back.

from geomagic.api.v3 import *

texture = Texture()
texture.setSize(512, 512, 3)
texture.fill(GeoColor(0.2, 0.2, 0.5))
texture.setColor(5, 5, GeoColor(1.0, 0.0, 0.0))
color55 = texture.color(5, 5)
color = texture.color(6, 6)
print("color55=" + str(color55))
print("color="+str(color))
