SelectOutliers¶

Computes the outlying points in a Points object object and modifies a Points Selection.

Examples

Create a set of points with one outlier then run SelectOutliers to find it.

from geomagic.api.v3 import *

# Create a point set with one outlier
pts = createHeightMapOfPoints(10, lambda x,y : 0)
pts.addPoint(Vector3D(10,10,10))  # the outlier

# Create a point selection
selection = PointSelection(pts)
selection.selectAll()

# Run it
selector = SelectOutliers()
selector.selection = selection
selector.run()
numOutliers = selection.numSelected # expect to be 1
