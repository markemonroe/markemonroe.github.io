BestFitAlign¶

BestFitAlign attempts to align a set of points (A Points or GriddedPoints object) with a DataObject. Returns the transform that can be used to fit the points onto the object.

Examples

Create a sample reference mesh and a test point object that are not aligned and run BestFitAlign to align them.

Create a sample reference mesh and a test point object that are not aligned and run BestFitAlign to align them using fineAdjustmentsOnly

from geomagic.api.v3 import *

# Create sample mesh. This will be used as the fixedObject
refMesh = createRefinedTetrahedron()

# Create points from mesh. This will be used as the floatingObject
testPoints = createPointsFromMesh(refMesh)

# Transform the floatingObject a bit so that the two models
# need to be aligned.
rotvec = Vector3D(0, 0, 1)
rotmat = Transform3D()
rotmat.setRotationTransform(rotvec, 0.05)
testPoints.transform(rotmat)

# Align fixedObject and floatingObject using BestFitAlign.
bfAlign = BestFitAlign()
bfAlign.fineAdjustmentsOnly = True
bfAlign.tolerance = 0.01
bfAlign.sampleSize = 1500
bfAlign.autoDeviatorElimination = False
bfAlign.fixedObject = refMesh
bfAlign.floatingObject = testPoints
bfAlign.run()

# distance needed to move test to get it aligned to ref.
dist1 = bfAlign.distance
