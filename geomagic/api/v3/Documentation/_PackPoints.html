PackPoints¶

This modifier optimizes the memory used by a Points object. Normally when deleting points in the list, the positions of remaining points are not moved. This can results in the Points object still using the same amount of memory even if a large portion of the points have been deleted. PackPoints remaps the points and normals to only use the space needed for the existing points. After a pack any indices into the points you may have previously stored will be invalid.

Examples

Optimize the memory usage of a points object by packing it.

from geomagic.api.v3 import *
   
def heightFunc(x, y):
   return 1.0

# Create array of points
pts = createHeightMapOfPoints(10, heightFunc)

# Remember last point index
ptIndex = pts.numPoints - 1

pos1 = pts.getPosition(ptIndex)

# Remove all points except last.
for i in range(0, pts.numPoints - 1):
   pts.removePoint(i)

# Pack points
pack = PackPoints()
pack.points = pts
pack.run()

# PtIndex should be an invalid index after packing.
hitException = False
try:
   pts.getPosition(ptIndex)
except Exception as e:
   hitException = True
