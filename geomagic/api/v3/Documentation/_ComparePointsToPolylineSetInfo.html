ComparePointsToPolylineSetInfo¶

The result of a ComparePointsToPolylineSet run is stored in a ComparePointsToPolylineSetInfo object. Overall statistics of the comparison are available as attributes. For each point that was successfully projected onto one of the reference paths, several pieces of information can be queried, like its index in the original point set, its normal there, and the location, normal and path information of the closest point to it that was found on a reference path.

It is important to remember that the indices that get passed into these queries range up to size. They are not the indices of the original points. To get those, use the method getPointIndex.

Examples

Project points to a set of polylines.

from geomagic.api.v3 import *

numPoints = 101
pts = Points()

p1 = Polyline()
p1.initialize(numPoints)

p2 = Polyline()
p2.initialize(numPoints)

p3 = Polyline()
p3.initialize(numPoints)

perturbvec = Vector3D(0.0, 0.01, 0.0)
for i in range (0, numPoints, 1):
   x = i-50
   y = 0.5*x*x
   z = 0.0
   ptvec = Vector3D(x, y, z)
   p1.points [ i ] = ptvec
   pts.addPoint(ptvec + perturbvec)

   y = y + 50 # parallel parabola - DirIndex(1, 0)
   ptvec = Vector3D(x, y, z)
   p2.points [ i ] = ptvec
   pts.addPoint(ptvec + perturbvec)

   y = y + 50 # parallel parabola - DirIndex(2, 0)
   ptvec = Vector3D(x, y, z)
   p3.points [ i ] = ptvec
   pts.addPoint(ptvec + perturbvec)
   
polySet = PolylineSet()   
path1 = polySet.addPolyline(p1)
path2 = polySet.addPolyline(p2)
path3 = polySet.addPolyline(p3)

compare = ComparePointsToPolylineSet()
compare.polylineSet = polySet
compare.points = pts
compare.maxDeviation = 10.0

plane = Plane()
plane.initialize(Vector3D(0.0, 0.0, 0.0), Vector3D(0.0, 0.0, 1.0))
compare.plane = plane
compare.run()
compinfo = compare.comparisonResult
numPts = compinfo.size
# reference parabola normals point down - so we expect negative distances
for i in range(0,numPts,1):
    ptIx = compinfo.getPointIndex(i)
    pathId = compinfo.getReferencePath(i)
    dist = compinfo.getDistance(i)
    #print ("i="+str(i)+", ptIx="+str(ptIx)+", pathId="+str(pathId)+ ", dist="+str(dist))
 
maxdist = compinfo.maxDistance
mindist = compinfo.minDistance
avdist = compinfo.averageDistance
stddevdist = compinfo.stdDevDistance
