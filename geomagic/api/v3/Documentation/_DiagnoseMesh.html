DiagnoseMesh¶

Perform a diagnosis of the mesh for problems such as intersections, small holes, and spikes.

Examples

Load a sample mesh and diagnose the problems with it.

from geomagic.api.v3 import *

# Create a sample mesh
m = createTetrahedron()

# Diagnose the mesh
diagnosis = DiagnoseMesh(m)
diagnosis.run()

# Query the results
hasProblems = diagnosis.hasProblems
totalProblems = diagnosis.totalProblems
intersections = diagnosis.numIntersections
smallComponents = diagnosis.numSmallComponents
smallHoles = diagnosis.numSmallHoles
smallTunnels = diagnosis.numSmallTunnels
spikeEdges = diagnosis.numSpikeEdges
spikeVertices = diagnosis.numSpikeVertices

Create a custom repair strategy and diagnose the problems with a mesh according to that strategy.

from geomagic.api.v3 import *

# Create a sample mesh
m = createTetrahedron()

# Create a repair strategy
strategy = RepairStrategy()
strategy.intersections = False
strategy.smallComponents = False
strategy.smallHoles = False
strategy.smallTunnels = True
strategy.spikeEdges = False
strategy.spikeVertices = False

# Diagnose mesh for small tunnels
diagnosis = DiagnoseMesh(m)
diagnosis.strategy = strategy
diagnosis.run()

# If small tunnels are found, repair them
if diagnosis.hasProblems == True:
   repair = RepairMesh(m)
   repair.strategy = strategy
   repair.run()
