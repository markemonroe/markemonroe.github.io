ColorArray¶

This is a container for an internal array of color values. It can be iterated over, its length can be queried through len(), elements can be accessed by the array operator [], and the array can be extended using append and shortened using pop.

Examples

Create an array and visit its elements using an iterator.

from geomagic.api.v3 import *

# Create an array with 3 elements
arr = ColorArray()
arr.append (GeoColor (0.0, 0.1, 0.2))
arr.append (GeoColor (0.1, 0.2, 0.3))
arr.append (GeoColor (0.2, 0.3, 0.4))

# Visit all elements
for c in arr:
   print(c)
