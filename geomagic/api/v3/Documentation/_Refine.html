Refine¶

Performs mesh refinement using various techniques

If True, vertices can be moved during refinement to provide more uniform triangles, if False no existing vertices may be moved. Use in 3X and 4X option

Examples

Refine a sample mesh three times.

from geomagic.api.v3 import *

# Create a tetrhedron 
m = createTetrahedron()

# Refine it
mr = Refine(m)
mr.moveVertices = 1
mr.fixBoundaries = 1
mr.subdivideBy = mr.ThreeX
mr.run()

# Refine again
mr.subdivideBy = mr.FourX
mr.run()

# Refine again.
mr.subdivideBy = mr.ThreeX
mr.run()
m.pack()
