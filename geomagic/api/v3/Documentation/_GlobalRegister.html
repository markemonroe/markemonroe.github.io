GlobalRegister¶

GlobalRegister performs global registration of several models, represented as Points, GriddedPoints, or Mesh objects.

Examples

from geomagic.api.v3 import *

# Create three mesh to register together.
m1 = createRefinedTetrahedron()
m2 = createRefinedTetrahedron()
m3 = createRefinedTetrahedron()

# 1st model transform is identity. No need to transform.
xf1 = Transform3D()

# Transform 2nd model so that it is not aligned with the others.
xf2 = Transform3D()
xf2.setRotationTransform(Vector3D(0, 0, 1), 0.05)
m2.transform(xf2)

# Transform 3rd model so that it is not aligned with the others.
xf3 = Transform3D()
xf3.setRotationTransform(Vector3D(1, 0, 0), -0.03)
m3.transform(xf3)

# Create global registration.
globalReg = GlobalRegister()
globalReg.simultaneous = 1

# Add models. Specify the first model as the one to be pinned, the model
# that the 2nd and 3rd models should be aligned to. These return ids with
# which you can use to get the transforms that are the result of the registration.
id1 = globalReg.addModel(m1, True, xf1)
id2 = globalReg.addModel(m2, False, xf1)
id3 = globalReg.addModel(m3, False, xf1)
result = globalReg.run()

# These are the transforms computed by GlobalRegister for each mesh.
xf1out = globalReg.transform(id1)
xf2out = globalReg.transform(id2)
xf3out = globalReg.transform(id3)
