PolylineSet¶

Represents collection of connected directed paths of PolylineOnMesh elements or Polyline elements. Various modifiers can perform operations on polylines, or use them to analyze or modify mesh. Examples: sectioning, mesh subdivision, wireframe manipulation.

Examples

Create a PolylineSet and manipulate it.

from geomagic.api.v3 import *

# Create an empty PolylineSet
polySet = PolylineSet()
print('numPolylines = ' + str(polySet.numPolylines))

# Add first polyline  
polyline = Polyline()
polyline.initialize(3)

polyline.points[0] = Vector3D(0.0, 1.0, 1.0)
polyline.points[1] = Vector3D(0.0, 1.0, 2.0)
polyline.points[2] = Vector3D(0.0, 1.0, 3.0)

id1 = polySet.addPolyline(polyline)
print('numPolylines = ' + str(polySet.numPolylines))

# Add second polyline

polylineOnMesh = PolylineOnMesh()
polylineOnMesh.initialize(2)

polylineOnMesh.points[0] = Vector3D(0.0, 1.0, 1.0)
polylineOnMesh.points[1] = Vector3D(0.0, 2.0, 1.0)

polylineOnMesh.triIndexes[0] = 1
polylineOnMesh.triIndexes[1] = 2

id2 = polySet.addPolyline(polylineOnMesh)
print('numPolylines = ' + str(polySet.numPolylines))

for i in range(polySet.maxPolylineIndex+1):
   if polySet.isValidPolylineIndex(i):
      p = polySet.getPolyline(i)

      for idx in range(0, len(p.points)):
         print("point[", idx, "]= ", p.points[idx])
         print("triIndex[", idx, "]= ", p.triIndexes[idx])
