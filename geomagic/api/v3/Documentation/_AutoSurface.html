AutoSurface¶

Create untrimmed NURBS patches on a polygon model, and save them to an IGES file.

class AutoSurface¶

Bases: geomagic.api.v3.swig.Modifier

Attributes

Name

Read only

Type

Range

Default Value

Usage hints

Description

fileName

No

string

Input (required)

Name of the IGS file to create. Must have extension .igs or .iges

mesh

No

Mesh

None

Input (required)

Polygon mesh to be surfaced. If necessary, the mesh will be modified to have no non-manifold vertices. The mesh also has to be orientable.

adaptiveFit

No

bool

True

Input (optional)

Use tolerance based NURBS fitting. If this is disabled, the number of control points used in fitting is determined by the detail attribute.

autoMerge

No

bool

False

Input (optional)

Try to automatically merge NURBS patches after fitting. This can not be combined with the adaptiveFit setting.

detail

No

double

0.000000 - 1.000000

0.333333

Input (optional)

Detail level of the resulting NURBS patches. A higher detail level will use more control points.

duplicateMesh

No

bool

True

Input (optional)

Specify whether to duplicate mesh prior to autosurfacing.

extendContours

No

bool

True

Input (optional)

Extend contours when creating patch layout.

fixedEdges

No

EdgeSelection

None

Input (optional)

If fixed edges are specified, they will be used as patch boundaries.

fixedVertices

No

VertexSelection

None

Input (optional)

If fixed vertices are specified, they will be used as patch corners.

geometry

No

enumeration

Mechanical

Input (optional)

Either set to Mechanical or Organic, to optimize the operation for the type of geometry.

numPatches

No

int

0 - +INF

0

Input (optional)

Approximate number of NURBS patches to create. If numPatches is 0, the modifier will choose an appropriate number of patches automatically.

sharpenConstrainedContours

No

bool

False

Input (optional)

If set, internal boundaries of the polygon mesh will be converted to sharp edges on the NURBS body.

tolerance

No

double

0.000000 - +INF

0.000000

Input (optional)

Tolerance to observe when using the adaptiveFit setting for NURBS. If tolerance is 0, a default value is computed based on the size of the model.

units

No

enumeration

0 - 6

Millimeters

Input (optional)

Units to be used in the IGS file. Default is millimeters. Unit types are defined in Length

errorMsg

Yes

string

Query autosurfacing error message, if operation failed.
