Sphere¶

Represents a spherical feature.

Diagram

class Sphere¶

Bases: geomagic.api.v3.swig.SurfaceFeature

Events

Name

Description

deleted

Triggered immediately before a DataObject is deleted.

updated

Published immediately after a DataObject’s data has been updated.

property_modified

Triggered when a property is changed. Args contains an attribute called “property” which is the property that changed.

Attributes

Name

Read only

Type

Range

Default Value

Description

axis

No

Vector3D

[0.00 0.00 0.00]

3D Vector through the poles of the sphere. Used to define coordinate system.

boundingBox

Yes

GeoBounds

[-1.000000 -1.000000 -1.000000] [1.000000 1.000000 1.000000]

The 3D dimensional bounds of the surface.

center

No

Vector3D

[0.00 0.00 0.00]

3D Vector representing the position that is equidistant from every point on the sphere.

latitudeRange

No

Vector2D

[-1.57 1.57]

Represents the minimum and maximum extents of a partial sphere orthogonal to the axis.

longitudeRange

No

Vector2D

[0.00 6.28]

Represents the minimum and maximum extents of a partial sphere in the direction of the axis.

radDir

No

Vector3D

[0.00 0.00 0.00]

3D Vector from the center to the equator of the sphere perpendicular to the axis. Along with the axis can be used to compute the coordinate system of the sphere.

radius

No

double

0.000000 - +INF

1.000000

Distance from the center of the sphere to every point on the sphere.

solidity

No

enumeration

0 - 2

NonSize

Solidity of the feature. Can be Hollow, Solid, or NonSize.

Examples

Creates a sphere.

from geomagic.api.v3 import *


# Create Sphere
sphere = Sphere()
sphere.center = Vector3D(0.0, 0.0, 0.0)
sphere.radius = 0.5
sphere.axis = Vector3D(0.0, 0.0, 1.0)
