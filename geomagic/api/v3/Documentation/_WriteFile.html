WriteFile¶

WriteFile is a modifier that performs file writes. Mesh, Point or Gridded Point object should be specified as attributes to be written in file. .wrp(Wrap) files are not supported, use geoapp.saveFile to write .wrp files.

By default the WriteFile modifier examines the file extension and matches it to one of the supported file types. Examples

from geomagic.api.v3 import *

# get the location of the example data
path = os.path.abspath(os.path.dirname(__file__)) + "/../../../../../pyscripts/data/"

# create and initialize mesh, add some points, build the mesh
mb = CreateMesh()
v000 = mb.addPoint(Vector3D(0, 0, 0))
v100 = mb.addPoint(Vector3D(2, 0, 0))
v010 = mb.addPoint(Vector3D(0, 4, 0))
f1 = mb.addTriangle(v000, v100, v010)
mb.run()
m = mb.mesh

# create write object
filewriter = WriteFile()

# set path
filewriter.filename = path + "/singletriangle2.obj"

# set mesh within filewrite object
filewriter.mesh = m

# write the file
filewriter.run()
