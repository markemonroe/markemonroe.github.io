ComparePointsToPolylineSet¶

Compares a reference set of polylines on a mesh and a test points object. The data of both objects is expected to be in the same plane. The result is represented as a ComparePointsToPolylineSetInfo object.

This modifier is similar to ComparePointsToMesh, but it operates in a plane, not in full 3D.

Examples

Project points to a set of polylines.

from geomagic.api.v3 import *

numPoints = 101
pts = Points()

p1 = Polyline()
p1.initialize(numPoints)

p2 = Polyline()
p2.initialize(numPoints)

p3 = Polyline()
p3.initialize(numPoints)

perturbvec = Vector3D(0.0, 0.01, 0.0)
for i in range (0, numPoints, 1):
   x = i-50
   y = 0.5*x*x
   z = 0.0
   ptvec = Vector3D(x, y, z)
   p1.points [ i ] = ptvec
   pts.addPoint(ptvec + perturbvec)

   y = y + 50 # parallel parabola - DirIndex(1, 0)
   ptvec = Vector3D(x, y, z)         
   p2.points [ i ] = ptvec
   pts.addPoint(ptvec + perturbvec)

   y = y + 50 # parallel parabola -  DirIndex(2, 0)
   ptvec = Vector3D(x, y, z)
   p3.points [ i ] = ptvec
   pts.addPoint(ptvec + perturbvec)
# test points and reference wireframe are ready

polySet = PolylineSet()   

p = Polyline()
p.initialize(numPoints)
path1 = polySet.addPolyline(p1)
path2 = polySet.addPolyline(p2)
path3 = polySet.addPolyline(p3)

compare = ComparePointsToPolylineSet()
compare.polylineSet = polySet
compare.points = pts
compare.maxDeviation = 10.0

plane = Plane()
plane.initialize(Vector3D(0.0, 0.0, 0.0), Vector3D(0.0, 0.0, 1.0))
compare.plane = plane
compare.run()
# get 2D comparison result
compinfo = compare.comparisonResult
