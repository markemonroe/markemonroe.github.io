BestFitSphere¶

Creates a sphere based on a selection using Best Fit.

class BestFitSphere¶

Bases: geomagic.api.v3.swig.FeatureFitter

Attributes

Name

Read only

Type

Range

Default Value

Usage hints

Description

selection

No

Selection

None

Input (required)

TriangleSelection, PointSelection, or QuadSelection to be used for fitting.

contactFitting

No

bool

False

Input (optional)

False means that the surface will run in between, as an average of the points defining the surface/feature, True means it runs at the outside boundary of the set of the points (solidity determines which side the material is).

extentOutlier

No

double

-INF - +INF

0.001000

Input (optional)

0 <= extentOutlier < 1. This is used for establishing the extent of the feature. Height of cylinder/cone, size of rectangle of area of plane, etc. This will be the ratio of points at the upper and lower end tails that will be cut off as outliers to establish the size of the feature.

maxrad

No

double

-INF - +INF

1000.000000

Input (optional)

Maximum radius of the sphere that can be created.

robustFitIterations

No

int

-INF - +INF

2

Input (optional)

Set it zero if no automatic outlier removal is needed. If it is non-zero then it specifies the number robust fit iterations to be performed during the fitting. (See robustFitOutliers.)

robustFitOutliers

No

double

-INF - +INF

20.000000

Input (optional)

Percentage (0 <= ro < 100). For outlier removal in best fit algorithms the user can request robust fit iteration. This means that after a full best-fit least-squares fit, all the points are reevaluated as distance from the initial feature and the farthest ones of robustFitOutliers percentage are (temporarily) discarded. Then a refit is performed for the remaining points. This is done robustFitIterations times.

solidity

No

enumeration

-INF - +INF

NonSize

Input (optional)

If contactFitting is True this property specifies if the feature is considered a void (Hollow) or a ball (Solid), or a non-size feature (NonSize).

goodSelection

No

Selection

None

Output

It is the actual (outliers removed) selection (TriangleSelection, PointSelection, or QuadSelection) that has been used for the fitting.

resultFeature

No

Sphere

None

Output

The output Sphere.

Constants

Name

Description

Hollow

If solidity is Hollow then the feature will be considered a spherical void. The contact feature is a largest limiting feature that is enclosed by the selection.

NonSize

If solidity is NonSize then the orientation is determined by the normals or, if they do not exist, the feature will be considered solid.

Solid

If solidity is Solid then the feature will be considered a solid (a ball). The contact feature is a smallest limiting feature that contains the selection.

Examples

Fits a sphere to a spherical sample set of points.

from geomagic.api.v3 import *



# Create selection composed of a roughly spherical set of points.
center = Vector3D(0.0, 0.0, 0.0)
pts = createSphericalPoints(5.0, center);
ptSelection = PointSelection(pts)
ptSelection.selectAll()

# Set the selection and run the fitter.
bestFitSphere = BestFitSphere()
bestFitSphere.selection = ptSelection
bestFitSphere.run()

# Sphere created by fitter
sphere = bestFitSphere.resultFeature
