filterGriddedPointsByAngle

No

bool

True

If True, ignores points whose normal is greater than griddedPointsAngle. In other words, points are ignored if they do not “face” the scanner within a reasonable angle. This can result in cleaner data.

filterGriddedPointsByEdgeLength

No

bool

False

If True filters out any quads whose edge length is less then griddedPointsEdgeLength. If False then all quads are loaded.

filterGriddedPointsByRatio

No

bool

False

If True filters out any quads whose width is longer than its height or whose height is longer than its width by the given ratio griddedPointsRatio. This is so that very elongated quads that may not be viable data are not loaded. If False all quads are loaded regardless of ratio.

griddedPointsSampleRatio

No

int

1 - 50

True

Specifies the size of a sample in which one quad will be chosen. This ratio is the dimension of a rectangle of quads in which one quad will be loaded to represent the rectangle of quads. The approximate number of quads that are loaded are 1 over n squared. If the ratio is 4 then it means for each 4 by 4 group of quads one quad will be loaded. A ratio of 4 would mean about 1/16 of the quads in a file would be loaded.

Examples

from geomagic.api.v3 import *

# get the location of the example data
path = os.path.abspath(os.path.dirname(__file__)) + "/../../../../../pyscripts/data/"

openOptions = FileOpenOptions()
openOptions.samplePoints = True
openOptions.samplePercentage = 50.0

reader = ReadFile()
reader.options = openOptions
reader.filename = path + "/teapot-4.vtx"
reader.run()
