RepairSelection¶

Take an arbitrary TriangleSelection and prepare it for better fitting. Ensures the selection has no bad holes and has no non-manifold vertices.

Examples

Run RepairSelection on a random selection.

from geomagic.api.v3 import *


mesh = createRoundedBox(2)

# Pick a random triangle to expand from
randomTriangle = 123
selection = TriangleSelection()
selection.mesh = mesh
selection.setSelected(randomTriangle, True)

# Expand the selection based on crease angle.
creaseAngle = ExpandSelectionByCreaseAngle()
creaseAngle.mesh = mesh
creaseAngle.selection = selection
creaseAngle.creaseAngle = 4.0
creaseAngle.run()
selection = creaseAngle.selection
numSelected = selection.numSelected

repair = RepairSelection()
repair.selection = selection
repair.run()
