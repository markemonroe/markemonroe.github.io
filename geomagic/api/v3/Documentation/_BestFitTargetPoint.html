BestFitTargetPoint¶

Creates a TargetPoint using a selection.

When finding the contact with the model move a disc with diameter size until the disc intersects with the model. The contact point is the center of the disc.

When finding the contact with the model use a line intersection.

Examples

Fits a target point to a model.

from geomagic.api.v3 import *



# Create cone mesh
mesh = openCone()

# Create a triangle selection that 
triSelection = TriangleSelection()
triSelection.mesh = mesh
triSelection.selectAll()

# Create and run fitter.
bestFitTargetPoint = BestFitTargetPoint()
bestFitTargetPoint.selection = triSelection
bestFitTargetPoint.model = mesh
bestFitTargetPoint.size = 0.1
bestFitTargetPoint.contactArea = BestFitTargetPoint.ContactArea_Area
bestFitTargetPoint.onSurfacePoint = Vector3D(-0.142, 0.481, -0.460)
bestFitTargetPoint.outsideSurfacePoint = Vector3D(-0.142, 1.481, -0.460)
bestFitTargetPoint.run()

# Result target point feature.
targetPoint = bestFitTargetPoint.resultFeature
