GridIndex¶

The 2D index of a point in GriddedPoints. The s and t indices may be negative, though this is not common.

Examples

Create a GriddedPoints and do some simple navigation.

from geomagic.api.v3 import *

# Create a square 2 by 2 grid of quads.
def plane(x,y):
   return x + y
pts = createHeightMapOfGriddedPoints(2, plane)

# Start at the corner.
gridIndex = pts.getGridIndex(0)

# Move one to the right
gridIndex.incS()

# Move one down
gridIndex.incT()
