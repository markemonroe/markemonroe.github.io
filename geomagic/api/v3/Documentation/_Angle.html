Angle¶

Conversion of angle units. The angle unit used internally is radians.

class Angle¶

Bases: object

Constants

Name

Description

Degrees

1 degree is PI/180 = 0.0174532925 radians.

Radians

1 radians is 1 radians.

Methods

Degrees = 1¶
Radians = 0¶

Examples

Convert between radians and degree.

from geomagic.api.v3 import *
print("1 radian is %f degrees" % (Angle.convertTo (Angle.Degrees, 1.0)))
print("1 degree is %f radians" % (Angle.convertFrom (Angle.Degrees, 1.0)))
print("The abbreviation for %s is %s" % (Angle.name (Angle.Degrees), Angle.abbreviation (Angle.Degrees)))
