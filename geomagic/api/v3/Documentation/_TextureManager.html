TextureManager¶

Data object that contains mesh texture information, like texture coordinates and images. It can be used for various texture-related queries, as argument for texture-related operations. Multiple texture managers can be associated with a given mesh, but only one can have an active status, which is used for rendering. For getting and setting active texture manager please refer to GetActiveTextureManager and SetActiveTextureManager.

Examples

Perform some queries on texture of given mesh.

from geomagic.api.v3 import *

pth = os.path.dirname(os.path.abspath(__file__)) + "/../../../../../pyscripts/data"
filereader = ReadFile()
filereader.filename = pth + u"/coin.obj"
result = filereader.run()   
m = filereader.mesh

gettm = GetActiveTextureManager(m)
gettm.run()
tm = gettm.textureManager;

texelSize = tm.texelSize
print("texelSize="+str(texelSize))
numTextures = tm.numTextures
print("numTextures="+str(numTextures))
colorAt_12_12 = tm.color(1, 12, 12)
print("color at texture 1, 12:12")
print(colorAt_12_12.r())
print(colorAt_12_12.g())
print(colorAt_12_12.b())
