ReadFile¶

ReadFile is a modifier that performs file reads. Mesh, Point or Gridded Point object will be available as attribute upon a successful read. Objects are not automatically added to the current application. .wrp (Wrap) files are not supported, use geoapp.openFile to read .wrp files.

Examples

from geomagic.api.v3 import *

# get the location of the example data
path = os.path.abspath(os.path.dirname(__file__)) + "/../../../../../pyscripts/data"

print("path = " + path)
# create ReadFile object
reader = ReadFile()

# set path
reader.filename = path + "/axial.obj"

# read the file
reader.run()

# get mesh object
mesh = reader.mesh
