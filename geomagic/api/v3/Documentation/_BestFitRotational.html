BestFitRotational¶

Best fits a rotational surface to the selected triangular mesh.

If this is true, this forces the orientation of the profile (sketch) plane to be parallel with a World Coordinate Axis if the optimal axis of revolution has been within snapAngle radians of the coordinate axis. This will also force the sketch lines onto the other coordinate axes.

Examples

Creates a rotational surface.

from geomagic.api.v3 import *

import math

# Create a cylinder
cylinder = Cylinder()
cylinder.initialize(Vector3D(0.0, 0.0, 0.0), Vector3D(0.0, 0.0, 1.0), 5.0, 1.0)

# Create amd run modifier to convert cylinder to a mesh.
createMeshFromSurface = CreateMeshFromSurface()
createMeshFromSurface.surface = cylinder
createMeshFromSurface.run()

# Mesh that approximates the surface of the cylinder.
mesh = createMeshFromSurface.mesh

# Create axis aligned bounding box that contains part of the mesh.
bounds = GeoBounds(Vector3D(-2.0, -2.0, 1.0), Vector3D(2.0, 2.0, 4.0))

# Create a selection from the triangles whose centroid is inside the box.
selectTriangles = SelectTrianglesIn3DBox()
selectTriangles.mesh = mesh
selectTriangles.box = bounds
selectTriangles.run()

# Triangle selection created by modifier.
triSelection = selectTriangles.selection

#Create a fitter
fitter = BestFitRotational()
fitter.meshSelection = triSelection
fitter.fitAsLineArc = 1
fitter.run()

#get the surface
rotationalSurface = fitter.resultSurfaces[0]
sketch = fitter.resultCurves
